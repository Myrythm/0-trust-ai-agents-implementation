# Feature 4: `zta.audit` Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `zta/audit.py` — append-only JSONL audit log with a SHA-256 hash chain. Every state-changing action across the control plane calls `Audit.append(...)`; tampering with any line breaks the chain at `verify_chain()`.

**Architecture:** Single `Audit` class wrapping a JSONL file. `AuditEvent` is a Pydantic model. `append()` reads the last line to find `prev_hash`, builds a new event, computes `this_hash = sha256(prev_hash || canonical(event_without_this_hash))`, and appends. `read_all()` returns the events (skipping malformed lines). `verify_chain()` recomputes hashes and compares. There are **no** `update` or `delete` methods — the API surface is append-only by construction.

**Tech Stack:** `pydantic`, stdlib `hashlib` + `json`. No new deps.

## Contract (per spec section 3.3)

```python
class AuditEvent(BaseModel):
    event_id: str           # uuid4 hex
    ts: str                 # ISO 8601 UTC
    agent_id: str
    request_id: str
    action: str             # e.g. "tool.invoke:db_query"
    resource: str           # e.g. "sqlite://data.db"
    decision: str           # "allow" | "deny" | "pending_approval" | "error"
    reason: str
    prev_hash: str          # 64 hex; genesis = "0"*64 for the first event
    this_hash: str          # 64 hex; sha256(prev_hash || canonical(body without this_hash))


class Audit:
    def __init__(self, path: Path) -> None: ...           # creates file + parents if missing
    def append(self, *, agent_id: str, request_id: str, action: str,
               resource: str, decision: str, reason: str) -> AuditEvent: ...
    def read_all(self) -> list[AuditEvent]: ...
    def verify_chain(self) -> bool: ...
```

**Hash chain rule:**

```
this_hash = sha256(prev_hash_bytes || canonical(event_without_this_hash))
```

where `canonical(...)` = `json.dumps(event, sort_keys=True, separators=(",", ":"))`.

**Failure modes:**
- File path parent dir missing → `Audit` creates it on construction
- `verify_chain()` returns False (never raises) for any inconsistency
- `read_all()` skips malformed lines (does not raise)
- `Audit` has no `update` / `delete` / `clear` methods → API surface is append-only by construction

**Concurrency note (MVP):** Single-process. No file locking. Concurrent appends from multiple processes can race; the runtime is in-process and a single Audit instance is used per session, so this is not a concern in practice.

```python
"""Append-only audit log with SHA-256 hash chain.

Every state-changing action across the control plane calls
`Audit.append(...)`. Events are stored as one JSON object per line in
a JSONL file. Each event carries `prev_hash` (the predecessor's
`this_hash`, or `GENESIS_HASH` for the first) and `this_hash` =
`sha256(prev_hash || canonical(event_without_this_hash))`. Tampering
with any line breaks the chain at `verify_chain()`.
"""

from __future__ import annotations

import hashlib
import json
import uuid
from datetime import UTC, datetime
from pathlib import Path
from typing import Any

from pydantic import BaseModel, ConfigDict, Field

from zta.errors import AuditError

GENESIS_HASH = "0" * 64


def _canonical(payload: dict[str, Any]) -> bytes:
    """Stable byte representation used for hashing."""
    return json.dumps(payload, sort_keys=True, separators=(",", ":")).encode("utf-8")


class AuditEvent(BaseModel):
    """One audit event as stored in the JSONL log."""

    model_config = ConfigDict(extra="forbid")

    event_id: str = Field(default_factory=lambda: uuid.uuid4().hex)
    ts: str = Field(default_factory=lambda: datetime.now(UTC).isoformat())
    agent_id: str
    request_id: str
    action: str
    resource: str
    decision: str
    reason: str
    prev_hash: str = GENESIS_HASH
    this_hash: str = ""

    def compute_this_hash(self) -> str:
        body = self.model_dump(mode="json", exclude={"this_hash"})
        return hashlib.sha256(
            self.prev_hash.encode("utf-8") + _canonical(body)
        ).hexdigest()


class Audit:
    """Append-only JSONL audit log with hash-chained integrity."""

    def __init__(self, path: Path) -> None:
        self.path = path
        path.parent.mkdir(parents=True, exist_ok=True)
        if not path.exists():
            path.touch()

    def append(
        self,
        *,
        agent_id: str,
        request_id: str,
        action: str,
        resource: str,
        decision: str,
        reason: str,
    ) -> AuditEvent:
        prev_hash = self._last_hash()
        event = AuditEvent(
            agent_id=agent_id,
            request_id=request_id,
            action=action,
            resource=resource,
            decision=decision,
            reason=reason,
            prev_hash=prev_hash,
        )
        event.this_hash = event.compute_this_hash()
        with self.path.open("a", encoding="utf-8") as f:
            f.write(event.model_dump_json() + "\n")
        return event

    def read_all(self) -> list[AuditEvent]:
        events: list[AuditEvent] = []
        if not self.path.exists():
            return events
        for line in self.path.read_text(encoding="utf-8").splitlines():
            line = line.strip()
            if not line:
                continue
            try:
                raw = json.loads(line)
                events.append(AuditEvent.model_validate(raw))
            except (json.JSONDecodeError, ValueError):
                continue
        return events

    def verify_chain(self) -> bool:
        expected_prev = GENESIS_HASH
        for event in self.read_all():
            if event.prev_hash != expected_prev:
                return False
            if event.this_hash != event.compute_this_hash():
                return False
            expected_prev = event.this_hash
        return True

    def _last_hash(self) -> str:
        events = self.read_all()
        if not events:
            return GENESIS_HASH
        return events[-1].this_hash
```

**Spec coverage:** Spec section 3.3 lists `AuditEvent` (all 10 fields) + `Audit` (`__init__`, `append`, `read_all`, `verify_chain`). All present. Hash chain rule matches spec ("this_hash = sha256(prev_hash || canonical(event))"). Append-only enforced by absence of mutation methods (tested).

**Placeholders:** None.

**Type consistency:** `AuditEvent` is a Pydantic model with `extra="forbid"`. `Audit.append(...)` signature matches spec. `verify_chain() -> bool`, `read_all() -> list[AuditEvent]`.

**Security:**
- Hash chain uses canonical JSON (sort_keys + tight separators) — no field reordering attack
- `prev_hash` is included in `this_hash` input → chain links can be verified
- `verify_chain()` re-reads the file and recomputes; doesn't trust stored `this_hash` alone
- File permissions not set in this feature; rely on the file system. Production should `chmod 0600` the log; documented as a deferred hardening item.
