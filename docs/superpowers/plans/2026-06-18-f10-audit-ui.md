# Feature 10: Audit UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wrap the `/audit` route in a Jinja template that renders the audit log as a readable table. `/api/audit` keeps returning JSON (for future JS polling in F12). F11 will do the same for `/policy`.

**Architecture:** New `templates/audit.html` extends `base.html`. The `/audit` route calls the same `Audit` reader as `/api/audit`, then renders the template. `/api/audit` is unchanged. Chain-valid status is shown as a banner at the top.

**Tech Stack:** `jinja2`, FastAPI template helpers (already set up in F9). No new deps.

## Contract

**`GET /audit`** (HTML page) → renders `audit.html` with:
- `chain_valid` (bool)
- `chain_status` ("valid" | "tampered" — derived)
- `events` (list of dicts, newest first)
- `event_count` (int)

**`GET /api/audit`** (JSON) → unchanged from F7.

```html
{% extends "base.html" %}
{% block title %}ZTA Audit{% endblock %}
{% block content %}
<h1>Audit log</h1>

{% if chain_valid %}
  <div class="banner banner-ok">Chain valid · {{ event_count }} event(s)</div>

      </tr>
    {% endfor %}
  </tbody>
</table>
{% else %}
<p class="empty">No events yet. Use the <a href="/">chat</a> to generate one.</p>
{% endif %}
{% endblock %}
```

### Task 3: Modify `app.py` to render the audit page

**Files:**
- Modify: `app.py`

- [ ] **Step 1: Update the `/audit` route to render the template**

Find the existing `audit_view` route:

```python
    @app.get("/audit")
    async def audit_view() -> dict[str, object]:
        return await api_audit()
```

Replace it with:

```python
    @app.get("/audit", response_class=HTMLResponse)
    async def audit_view(request: Request) -> Response:
        audit = Audit(cfg.audit_path)
        events = audit.read_all()
        # newest first
        events_sorted = list(reversed(events))
        return templates.TemplateResponse(
            request,
            "audit.html",
            {
                "chain_valid": audit.verify_chain(),
                "chain_status": "valid" if audit.verify_chain() else "tampered",
                "events": [e.model_dump(mode="json") for e in events_sorted],
                "event_count": len(events),
            },
        )
```
